Expressões e Operadores

Expressões são combinações de valores, variáveis e operadores que produzem um resultado.

Operadores Aritméticos

Realizam operações matemáticas:

var a: Inteiro = 10; var b: Inteiro = 3; var soma: Inteiro = a + b; // 13 var subtracao: Inteiro = a - b; // 7 var multiplicacao: Inteiro = a * b; // 30 var divisao: Real = a / b; // 3.33... var modulo: Inteiro = a % b; // 1 (resto da divisão)

Operadores de Comparação

Comparam valores e retornam verdadeiro ou falso:

var x: Inteiro = 5; var y: Inteiro = 10; var igual: Logico = x == y; // falso var diferente: Logico = x != y; // verdadeiro var menor: Logico = x < y; // verdadeiro var menorIgual: Logico = x <= y; // verdadeiro var maior: Logico = x > y; // falso var maiorIgual: Logico = x >= y; // falso

Operadores Lógicos

Combinam expressões lógicas:

E Lógico (e)

var idade: Inteiro = 25; var temCarteira: Logico = verdadeiro; var podeDirigir: Logico = idade >= 18 e temCarteira; // verdadeiro apenas se ambas condições forem verdadeiras

OU Lógico (ou)

var ehFimDeSemana: Logico = verdadeiro; var ehFeriado: Logico = falso; var podeDescansar: Logico = ehFimDeSemana ou ehFeriado; // verdadeiro se pelo menos uma condição for verdadeira

NÃO Lógico (!)

var chovendo: Logico = falso; var podeSair: Logico = !chovendo; // verdadeiro var estaChovendo: Logico = verdadeiro; var naoChuva: Logico = !estaChovendo; // falso

Operadores Unários

Operam em um único valor:

var numero: Inteiro = 42; var negativo: Inteiro = -numero; // -42 var condicao: Logico = verdadeiro; var inverso: Logico = !condicao; // falso

Operador de Atribuição

Atribui valores às variáveis:

var nome: Texto; nome = "João"; // Atribuição simples var contador: Inteiro = 0; contador = contador + 1; // Incremento

Acesso a Propriedades e Arrays

Acesso a Propriedades

var pessoa: Pessoa = nova Pessoa(); pessoa.nome = "Maria"; var nomeCompleto: Texto = pessoa.nome + " Silva";

Acesso a Arrays

var numeros: Lista = [1, 2, 3, 4, 5]; var primeiro: Inteiro = numeros[0]; // 1 var segundo: Inteiro = numeros[1]; // 2 numeros[0] = 10; // Modificando o primeiro elemento

Acesso Bidimensional

var matriz: Lista = [[1, 2], [3, 4]]; var elemento: Inteiro = matriz[0][1]; // 2 matriz[1][0] = 99; // Modificando elemento [1][0]

Precedência de Operadores

Ordem de avaliação (do maior para o menor):

  • Parênteses: ()
  • Acesso: . e []
  • Unários: ! e -
  • Multiplicação/Divisão: *, /, %
  • Adição/Subtração: +, -
  • Comparação: <, <=, >, >=
  • Igualdade: ==, !=
  • E lógico: e
  • OU lógico: ou
  • Atribuição: =

Exemplos Práticos

Cálculo Complexo

var a: Real = 10.0; var b: Real = 5.0; var c: Real = 2.0; var resultado: Real = (a + b) * c / (a - b); // (10 + 5) * 2 / (10 - 5) = 15 * 2 / 5 = 6

Validação Complexa

var idade: Inteiro = 20; var temRenda: Logico = verdadeiro; var scoreCredito: Inteiro = 750; var aprovado: Logico = (idade >= 18 e idade <= 65) e temRenda e scoreCredito >= 600;
