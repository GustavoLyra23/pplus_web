Classes

Classes são modelos para criar objetos que agrupam dados (atributos) e comportamentos (métodos).

Declaração de Classes

Use a palavra-chave classe seguida do nome da classe:

Classe Simples

classe Pessoa { var nome: Texto; var idade: Inteiro; funcao apresentar() { escrever("Olá, eu sou " + this.nome); } }

Criação de Objetos

Use a palavra-chave nova para criar instâncias da classe:

var pessoa1: Pessoa = nova Pessoa(); pessoa1.nome = "João"; pessoa1.idade = 25; pessoa1.apresentar();

Herança

Use estende para criar uma classe que herda de outra:

classe Estudante estende Pessoa { var curso: Texto; var matricula: Inteiro; funcao estudar() { escrever(this.nome + " está estudando " + this.curso); } }

Usando a Herança

var estudante: Estudante = nova Estudante(); estudante.nome = "Maria"; // Herdado de Pessoa estudante.idade = 20; // Herdado de Pessoa estudante.curso = "Informática"; estudante.apresentar(); // Método herdado estudante.estudar(); // Método próprio

Implementação de Interfaces

Use implementa para fazer a classe seguir um contrato de interface:

classe Retangulo implementa Forma { var largura: Real; var altura: Real; funcao calcularArea(): Real { return this.largura * this.altura; } }

Múltiplas Interfaces

Uma classe pode implementar várias interfaces:

classe Quadrado implementa Forma, Desenhavel { var lado: Real; funcao calcularArea(): Real { return this.lado * this.lado; } funcao desenhar() { escrever("Desenhando um quadrado"); } }

Palavra-chave this

Use this para referenciar o objeto atual dentro da classe:

classe Contador { var valor: Inteiro = 0; funcao incrementar() { this.valor = this.valor + 1; } funcao obterValor(): Inteiro { return this.valor; } }
