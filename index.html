Portugol++

Portugol++ é uma linguagem de programação interpretada focada para quem está aprendendo a programar e quer criar programas de forma fácil e rápida em português.

Seguindo os padrões da orientação a objetos, a linguagem é baseada em classes e objetos. (Se você não sabe o que é isso, não se preocupe)

A linguagem é inspirada no Portugol, uma linguagem de programação criada pela Fundação Bradesco para ensinar programação de forma fácil e acessível.

Características

  • Sintaxe simples e intuitiva
  • Fácil de aprender e usar
  • Open source
  • Várias bibliotecas disponíveis
