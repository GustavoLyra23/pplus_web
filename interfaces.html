Interfaces

Interfaces definem contratos que as classes devem seguir, especificando quais métodos devem ser implementados.

Declaração de Interfaces

Use a palavra-chave interface seguida do nome e assinaturas dos métodos:

Interface Simples

interface Forma { funcao calcularArea(): Real; funcao calcularPerimetro(): Real; }

Implementação de Interfaces

Classes que implementam uma interface devem fornecer todos os métodos declarados:

Classe Implementando Interface

classe Circulo implementa Forma { var raio: Real; funcao calcularArea(): Real { return 3.14159 * this.raio * this.raio; } funcao calcularPerimetro(): Real { return 2 * 3.14159 * this.raio; } }

Múltiplas Interfaces

Uma classe pode implementar várias interfaces separadas por vírgula:

Múltiplas Interfaces

interface Desenhavel { funcao desenhar(); funcao apagar(); }

Classe com Múltiplas Interfaces

classe Retangulo implementa Forma, Desenhavel, Colorivel { var largura: Real; var altura: Real; var cor: Texto; // Métodos da interface Forma funcao calcularArea(): Real { return this.largura * this.altura; } funcao calcularPerimetro(): Real { return 2 * (this.largura + this.altura); } // Métodos da interface Desenhavel funcao desenhar() { escrever("Desenhando retângulo"); } funcao apagar() { escrever("Apagando retângulo"); } // Métodos da interface Colorivel funcao definirCor(cor: Texto) { this.cor = cor; } funcao obterCor(): Texto { return this.cor; } }

Vantagens das Interfaces

  • Garantem que classes implementem métodos específicos
  • Permitem polimorfismo
  • Facilitam a organização do código
  • Tornam o código mais flexível e reutilizável

Exemplo Prático

// Usando as classes que implementam Forma var circulo: Circulo = nova Circulo(); circulo.raio = 5.0; var retangulo: Retangulo = nova Retangulo(); retangulo.largura = 10.0; retangulo.altura = 6.0; // Ambos têm os métodos da interface Forma escrever("Área do círculo: " + circulo.calcularArea()); escrever("Área do retângulo: " + retangulo.calcularArea());
