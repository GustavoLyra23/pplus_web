Funções

Funções são blocos de código reutilizáveis que executam uma tarefa específica e podem retornar um valor.

Declaração de Funções

Use a palavra-chave funcao seguida do nome, parâmetros e tipo de retorno:

Função Simples

funcao saudar() { escrever("Olá, mundo!"); }

Função com Parâmetros

funcao saudarPessoa(nome: Texto) { escrever("Olá, " + nome + "!"); }

Função com Retorno

funcao somar(a: Inteiro, b: Inteiro): Inteiro { return a + b; }

Parâmetros

Parâmetros são valores que a função recebe para processar:

Múltiplos Parâmetros

funcao calcularMedia(nota1: Real, nota2: Real, nota3: Real): Real { return (nota1 + nota2 + nota3) / 3; }

Parâmetros de Diferentes Tipos

funcao criarPerfil(nome: Texto, idade: Inteiro, ativo: Logico): Texto { return nome + " tem " + idade + " anos"; }

Chamada de Funções

Para executar uma função, use seu nome seguido dos argumentos:

// Chamadas das funções declaradas acima saudar(); saudarPessoa("Maria"); var resultado: Inteiro = somar(5, 3); var media: Real = calcularMedia(8.5, 7.0, 9.2);

Return

Use return para retornar um valor da função:

funcao ehPar(numero: Inteiro): Logico { return numero % 2 == 0; } funcao obterMaior(a: Inteiro, b: Inteiro): Inteiro { se (a > b) { return a; } senao { return b; } }
