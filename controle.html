Estruturas de Controle

Estruturas de controle permitem alterar o fluxo de execução do programa baseado em condições ou repetições.

Estrutura Condicional

Se / Senão

Execute código baseado em uma condição:

se (idade >= 18) { escrever("Maior de idade"); } senao { escrever("Menor de idade"); }

Se sem Senão

se (temperatura > 30) { escrever("Está muito quente!"); }

Condições Aninhadas

se (nota >= 90) { escrever("Excelente!"); } senao { se (nota >= 70) { escrever("Bom!"); } senao { escrever("Precisa melhorar."); } }

Estruturas de Repetição

Enquanto

Repete enquanto a condição for verdadeira:

var contador: Inteiro = 0; enquanto (contador < 5) { escrever("Contador: " + contador); contador = contador + 1; }

Para

Loop com inicialização, condição e incremento:

para (var i: Inteiro = 0; i < 10; i = i + 1) { escrever("Número: " + i); }

Faça-Enquanto

Executa pelo menos uma vez, depois verifica a condição:

var numero: Inteiro; faca { numero = lerInteiro("Digite um número positivo: "); } enquanto (numero <= 0);

Controle de Fluxo

Quebra

Sai do loop atual:

para (var i: Inteiro = 0; i < 100; i = i + 1) { se (i == 50) { quebra; // Sai do loop quando i for 50 } escrever(i); }

Continue

Pula para a próxima iteração do loop:

para (var i: Inteiro = 0; i < 10; i = i + 1) { se (i % 2 == 0) { continue; // Pula números pares } escrever("Número ímpar: " + i); }

Exemplos Práticos

Validação de Entrada

var senha: Texto; var tentativas: Inteiro = 0; enquanto (tentativas < 3) { senha = lerTexto("Digite a senha: "); se (senha == "123456") { escrever("Acesso liberado!"); quebra; } senao { tentativas = tentativas + 1; escrever("Senha incorreta. Tentativa " + tentativas); } }

Menu de Opções

var opcao: Inteiro; faca { escrever("1 - Novo jogo"); escrever("2 - Carregar jogo"); escrever("3 - Sair"); opcao = lerInteiro("Escolha uma opção: "); se (opcao == 1) { escrever("Iniciando novo jogo..."); } senao { se (opcao == 2) { escrever("Carregando jogo..."); } } } enquanto (opcao != 3);
