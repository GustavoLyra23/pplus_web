Importações

O sistema de importações permite usar código de outros arquivos e bibliotecas em seu programa.

Sintaxe Básica

Use a palavra-chave importar seguida do caminho do arquivo entre aspas:

importar "biblioteca.pp"; importar "utils/helpers.pp"; importar "matematica/calculos.pp";

Importação de Bibliotecas Padrão

Bibliotecas do sistema podem ser importadas diretamente:

importar "std/io.pp"; // Entrada e saída importar "std/math.pp"; // Funções matemáticas importar "std/string.pp"; // Manipulação de texto importar "std/list.pp"; // Operações com listas importar "std/file.pp"; // Manipulação de arquivos

Importação de Arquivos Locais

Importe arquivos do seu projeto usando caminhos relativos:

Estrutura de Projeto

projeto/
├── main.pp
├── classes/
│ ├── Pessoa.pp
│ └── Estudante.pp
└── utils/
├── matematica.pp
└── validacoes.pp

Importando no main.pp

importar "classes/Pessoa.pp"; importar "classes/Estudante.pp"; importar "utils/matematica.pp"; importar "utils/validacoes.pp";

Exemplo de Arquivo para Importação

Arquivo: utils/matematica.pp

// Funções matemáticas úteis funcao calcularPotencia(base: Real, expoente: Inteiro): Real { var resultado: Real = 1.0; para (var i: Inteiro = 0; i < expoente; i = i + 1) { resultado = resultado * base; } return resultado; } funcao calcularFatorial(n: Inteiro): Inteiro { se (n <= 1) { return 1; } return n * calcularFatorial(n - 1); }

Arquivo: classes/Pessoa.pp

classe Pessoa { var nome: Texto; var idade: Inteiro; funcao apresentar() { escrever("Olá, eu sou " + this.nome + " e tenho " + this.idade + " anos"); } funcao ehMaiorIdade(): Logico { return this.idade >= 18; } }

Usando Código Importado

Arquivo: main.pp

importar "classes/Pessoa.pp"; importar "utils/matematica.pp"; funcao main() { // Usando classe importada var pessoa: Pessoa = nova Pessoa(); pessoa.nome = "João"; pessoa.idade = 25; pessoa.apresentar(); // Usando funções importadas var potencia: Real = calcularPotencia(2.0, 3); var fatorial: Inteiro = calcularFatorial(5); escrever("2³ = " + potencia); escrever("5! = " + fatorial); }

Boas Práticas

Organização

  • Coloque todas as importações no início do arquivo
  • Organize arquivos em pastas por funcionalidade
  • Use nomes descritivos para arquivos e pastas

Convenções

  • Termine arquivos Portugol++ com .pp
  • Use PascalCase para nomes de classes
  • Use camelCase para funções e variáveis
  • Uma classe por arquivo (quando possível)

Importações Múltiplas

Você pode importar vários arquivos relacionados:

// Importando todo o módulo de geometria importar "geometria/Circulo.pp"; importar "geometria/Retangulo.pp"; importar "geometria/Triangulo.pp"; importar "geometria/utils.pp";

Evitando Conflitos

Se dois arquivos definem o mesmo nome, o último importado terá precedência:

importar "arquivo1.pp"; // Define funcao calcular() importar "arquivo2.pp"; // Também define funcao calcular() // calcular() usará a versão do arquivo2.pp
