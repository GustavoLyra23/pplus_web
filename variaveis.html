Variáveis

Variáveis são espaços na memória para armazenar dados que podem ser modificados durante a execução do programa.

Declaração de Variáveis

Use a palavra-chave var seguida do nome da variável:

Declaração com Tipo Explícito

var nome: Texto; var idade: Inteiro; var altura: Real;

Declaração com Inicialização

var nome: Texto = "Maria"; var idade: Inteiro = 30; var ativo: Logico = verdadeiro;

Declaração com Inferência de Tipo

O tipo pode ser inferido automaticamente pelo valor inicial:

var nome = "João"; // Inferido como Texto var idade = 25; // Inferido como Inteiro var altura = 1.80; // Inferido como Real

Atribuição de Valores

Use o operador = para atribuir valores às variáveis:

nome = "Pedro"; idade = idade + 1; ativo = falso;

Escopo de Variáveis

Variáveis declaradas dentro de funções são locais. Variáveis declaradas em classes são atributos da classe.

Variável Local

funcao exemploFuncao() { var local: Texto = "Só existe aqui"; }

Atributo de Classe

classe MinhaClasse { var atributo: Inteiro = 10; }
