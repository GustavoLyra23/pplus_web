Tipos de Dados

O Portugol++ oferece tipos de dados primitivos e compostos para armazenar diferentes tipos de informações.

Tipos Primitivos

Inteiro

Armazena números inteiros (sem casas decimais).

var idade: Inteiro = 25;

Real

Armazena números com casas decimais.

var altura: Real = 1.75;

Texto

Armazena sequências de caracteres (strings).

var nome: Texto = "João Silva";

Logico

Armazena valores booleanos (verdadeiro ou falso).

var ativo: Logico = verdadeiro;

Tipos Compostos

Lista

Armazena uma coleção ordenada de elementos.

var numeros: Lista = [];

Mapa

Armazena pares chave-valor (dicionário).

var dados: Mapa = [[ ] ];

Tipos Personalizados

Você também pode usar classes como tipos:

var usuario: Usuario = nova Usuario();
